# E-Commerce Sales & Customer Insights Dashboard 🛒

An interactive Power BI dashboard designed to track retail performance, analyze sales trends, and monitor key performance indicators (KPIs) for an online marketplace.

## 🚀 Project Overview
This project transforms raw e-commerce transactional data into actionable business intelligence. It provides a 360-degree view of sales health, helping stakeholders monitor revenue growth, shipping efficiency, and regional market penetration.

## 🛠️ Tech Stack
- **Power BI Desktop**: For data modeling, DAX development, and UI/UX design.
- **Power Query**: For data cleaning, handling null values, and ETL processes.
- **DAX (Data Analysis Expressions)**: To calculate complex metrics such as Year-over-Year (YoY) growth, Profit Margins, and Average Order Value (AOV).

## 📊 Key Features & Analysis
- **Executive Summary**: High-level KPIs including Total Sales, Total Profit, and Quantity Sold.
- **Temporal Analysis**: Monthly and quarterly sales trends to identify peak seasonal performance.
- **Geographic Insights**: Sales distribution across different regions and cities to pinpoint high-growth markets.
- **Category & Product Performance**: Deep dive into top-selling product categories and sub-categories.
- **Shipping & Logistics**: Analysis of shipping modes and delivery timelines.

## 📈 Dashboard Highlights
- **Interactive Filtering**: Dynamic slicers for Region, Segment, and Year.
- **Profitability Analysis**: Identification of loss-making products and regions to optimize business strategy.
- **Customer Segmentation**: Breakdown of sales by Consumer, Corporate, and Home Office segments.

## 📂 Repository Structure
- `ecommerce dashboard.pbix`: The core Power BI dashboard file.
- `/data`: Folder containing the raw sales data (CSV/Excel).
- `/media`: Screenshots and GIFs of the interactive dashboard.
